## Prerequisites
- CMake 3.16+
- Qt 6 (Qt Widgets)
- C++17-compatible compiler (MinGW/GCC on Windows)

## Proof of Concept (PoC)
A minimal Qt-based Tetris PoC: single block falling on a 10x20 grid.

## Run
```powershell
cmake -S . -B build
cmake --build build --target build_and_run
```
